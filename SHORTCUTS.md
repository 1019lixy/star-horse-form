# DynamicForm 表单设计器 - 快捷键说明

## 快捷键列表

### 编辑操作

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Ctrl + C` | 复制 | 复制选中的组件到剪贴板 |
| `Ctrl + X` | 剪切 | 剪切选中的组件到剪贴板 |
| `Ctrl + V` | 粘贴 | 粘贴剪贴板中的组件 |
| `Ctrl + Z` | 撤销 | 撤销上一步操作 |
| `Ctrl + Y` | 重做 | 重做上一步操作（也可用 `Ctrl + Shift + Z`） |
| `Ctrl + Shift + Z` | 重做 | 重做上一步操作（也可用 `Ctrl + Y`） |
| `Delete` | 删除 | 删除选中的组件 |
| `Ctrl + A` | 全选 | 全选所有组件 |
| `Ctrl + Shift + D` | 清空 | 清空所有组件 |
| `Ctrl + E` | 修改属性 | 批量修改属性 |

## 注意事项

- **输入框中自动放行**：当焦点在输入框（input、textarea、select、contenteditable）中时，快捷键自动放行给浏览器处理，不会拦截。这意味着在弹窗或属性面板的输入框中可以正常使用 Ctrl+C/V/X 等标准文本操作
- **弹窗时自动禁用**：打开对话框（代码生成、预览、批量编辑、数据源配置等）时快捷键自动禁用，关闭后自动恢复
- **双重保护机制**：即使弹窗未正确设置禁用标志，只要焦点在输入元素内，快捷键也不会被拦截
- 快捷键使用 `capture` 阶段监听，匹配快捷键时会拦截浏览器的默认行为（如 `Ctrl + S` 不会触发浏览器保存对话框）
